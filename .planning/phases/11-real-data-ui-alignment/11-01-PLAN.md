# Plan 11-01: Real Data Ingestion & UI Resemblance Alignment

## Objective
Thực hiện đồng bộ hóa dữ liệu thật từ database Supabase lên môi trường Vercel production, căn chỉnh toàn bộ giao diện UI/UX của trang chủ và trang so sánh lãi suất sang tông màu Emerald hiện đại kết hợp font chữ Instrument Sans sang trọng, đồng thời hoàn thiện bộ lọc so sánh sản phẩm nâng cao để đạt độ tương đồng tối đa với nền tảng mẫu NganHang.com.

## Requirements Addressed
- **Success Criteria 1:** Trang production https://bankng-prod.vercel.app/ hiển thị đầy đủ 28 ngân hàng và 134 bản ghi lãi suất thật từ Supabase database.
- **Success Criteria 2:** Giao diện trang chủ và trang so sánh lãi suất được căn chỉnh đồng bộ theo tông màu Emerald và font chữ Instrument Sans chuyên nghiệp giống hệt NganHang.com.
- **Success Criteria 3:** Hoàn thiện bộ lọc nâng cao theo nhóm ngân hàng và kỳ hạn trên các trang so sánh `/compare/[category]`.
