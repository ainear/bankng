# PROJECT.md

## Project

Bankng is a fintech comparison and lead-generation platform inspired by Nganhang.com.
It is being built as a fullstack monorepo with:

- public SEO web
- admin backoffice
- banker portal
- PostgreSQL/Prisma data layer
- calculators
- lead engine
- content and trust operations

## Current Workflow Context

This repository began under project-scope OMX planning in `.omx/`.
GSD has been bootstrapped in project scope to support structured phase verification and autonomous continuation from the current implementation state.

## Current Milestone

Establish the production-oriented foundation of the platform and complete core admin catalog/rates workflows before expanding broader public and banker product surfaces.

## Completed So Far

1. Foundation monorepo scaffold
2. DB schema baseline and Supabase sync
3. Admin CRUD for banks, categories, products, variants, rates, branches
4. Admin route protection and mutation feedback
5. Rate verification workflow and freshness indicators

## Near-Term Focus

- public comparison and detail flows
- banker portal expansion
- lead CRM and routing
- content/SEO system
- trust, moderation, and operational hardening
